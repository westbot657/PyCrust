# PyCrust
A sandboxed Python to wasm32-wasi compiler written in rust


Feature checklist
- [ ] Correctly lex and parse python
- [ ] Generate wasm-compatible byte code
- [ ] python source imports
- [ ] rust module imports
- [ ] Call host functions
- [ ] export functions to host
- [ ] Typed structs
- [ ] Dynamic types
- [ ] Classes
- [ ] Generators
- [ ] Reflection
- [ ] Additional magic methods
